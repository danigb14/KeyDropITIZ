# Banners_FastPass

Coloca aquí las imágenes que deseas mostrar en el carrusel de la página principal.

- Soporta archivos `.jpg`, `.jpeg`, `.png` y `.gif`.
- Al recargar la página se seleccionarán 3 imágenes al azar de este directorio.
- Nombres arbitrarios están bien; el código las importa dinámicamente.

Ejemplos:

```
src/assets/Banners_FastPass/banner1.jpg
src/assets/Banners_FastPass/banner-verano.png
src/assets/Banners_FastPass/promo.gif
```
